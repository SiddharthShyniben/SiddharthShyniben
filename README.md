Sid here. I’ve been breaking things for 8+ years to see how they work. I thrive at the soft spot between building cool stuff and learning new things.

I tend to start tons of projects and leave them as experiments rather than "finishing" them. I've been described as a  ["a god level developer who doesn't build projects"](./od.png)—my Github is mostly a graveyard of curiosity. I’m currently focusing on academic foundations, so most of this is "as-is" snapshots of various learning phases.

I’m primarily a JavaScript person ([certified >99% of pros on CodinGame](https://www.codingame.com/certification/FMBamJBEWhb2f4UOV8pGFQ)) but I’ve used TypeScript, Angular, Svelte, HTM/CSS, Node, Deno, and touched bits of C, Java, Rust, and Go.

I love code golf - I create [hyper-compressed programs](https://github.com/nuff-said) for fun. I am an avid vimmer and I love spending time tuning my dotfiles. I also blogged for a few years on [dev.to](https://dev.to/SiddharthShyniben).
